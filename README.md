# 🎙️ Voice Assistant (Python Project)

## 📌 Objective
The objective of this project is to build a simple voice assistant that can recognize user speech and respond to basic commands like greeting, telling time, and date.

---

## ⚙️ Tools & Technologies Used
- Python
- SpeechRecognition library
- PyAudio (for microphone input)
- datetime module (built-in)

---

## 🚀 Steps Performed
1. Installed required libraries for speech recognition.
2. Initialized the recognizer to capture voice input.
3. Used microphone to take user commands.
4. Converted speech into text using Google Speech API.
5. Processed commands using conditional statements.
6. Displayed responses like time, date, or greetings.
7. Added basic error handling for better performance.

---

## ✅ Outcome
The project successfully performs basic voice recognition tasks and responds to user commands. It demonstrates how speech input can be used to build interactive Python applications.

---

## 💡 Features
- Recognizes voice commands
- Tells current time and date
- Simple and beginner-friendly
- Handles basic errors
